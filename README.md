# Fleet-Analyser

Hoi4 Total War mod carrier battles simulator  

# Introduction

This program allows to simulate Hoi4 aircraft carrier battles with ease.  
Now, you know how carrier that you are creating in 1936 will perform in 1941 against your enemy.  

# Instalation

Download ZIP, unpack it and run it with C++ compiler of your choice.

# How to use

## Spreadsheet

Before going to program itself, there is a spreadsheet made with some gfx that show simplified calculations made by program.  
You can plug in your numbers shown in game in the panel and see the simplified results.

## Program

In your unpacked folder there are 2 .txt files. These are inputs.  
In them you can find modifiers names. You can edit them, but every description must end with "=".  
After "=", there is a number that is your modifier. File have some of them filled for demonstration.  

After you changed the inputs to ones you want, go to the compiler and run the program.  
In the console, there will be information about current hour, day/night cycle, information if any crits were made, and who is doing what.  
Below that, there will be calculation results of various phases, finished with damage done to booth sides of the battle.

Enjoy your testing!
